# Daily Views

### 以日历的形式显示你之前创建过的日记。
![preview](https://github.com/fatevase/SiYuanDailyView/blob/main/preview.png)

### 功能
* 显示对应笔记下的日记。
* 检索日记中的标签并显示前三个标签。
* 选择显示的主。笔记
